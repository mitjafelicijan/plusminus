# Tiny X11 Window Manager

## Requirements

- C compiler
- GNU Make
- pkg-config

X11 and Freetype development libraries are required.

```sh
sudo xbps-install libX11-devel freetype-devel
```

## Compilation

```sh
CC=clang make
```
